# Manual Screen #3 — 103 Carter Road, Nambour

## Executive summary
Initial screen outcome: **Hold / verify**

Reason:
- The site is credible enough to screen further because it is a real Sunshine Coast listing with a visible address, moderate lot size, and a positive access signal from two driveways.
- However, the listing does not provide the same explicit development signal as 39 Park Rd, and the decisive planning controls remain unverified in this pass.

## What is encouraging
- Sunshine Coast LGA location: yes
- Nambour suburb is relevant to the MVP search set
- **756sqm** site area is useful at screen level
- two driveways are a positive access/frontage signal
- enough public detail exists to justify structured screening

## What prevents a Proceed outcome
The following material checks remain unresolved:
1. current zone confirmation from council mapping/site report
2. local plan / precinct confirmation
3. minimum lot size / subdivision threshold confirmation
4. flood overlay check
5. steep land / landslide check
6. biodiversity / waterways check
7. prior application history check

## Current assessment view
This site sits between the first two screens:
- stronger raw site size and access signal than a weak control site
- weaker explicit development/planning signal than 39 Park Rd
- much smaller and less obviously strategic than 16 Mons Road

That makes it a useful middle-band calibration case for the MVP.

## Recommendation
- **Current recommendation:** Hold / verify
- **Why:** enough is positive to keep it in play, but not enough is verified to support a stronger proceed view
- **Commercial usefulness:** strong as a calibration case because it tests how the MVP handles a plausible but less clearly marketed candidate

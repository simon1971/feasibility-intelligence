# Manual Screen #4 — 11 School Road, Bli Bli

## Executive summary
Initial screen outcome: **Hold / verify**

Reason:
- The site has enough area to remain plausible at a high level, but the listing itself does not present a strong subdivision or redevelopment signal.
- In this pass, the decisive planning and overlay checks remain unresolved.

## What is encouraging
- Sunshine Coast LGA location: yes
- Bli Bli suburb is within MVP scope
- **855sqm** lot size is a positive screen-level input
- real listing with visible address and enough public detail for testing

## What weakens the case compared with the stronger candidates
- no verified zoning claim
- no explicit redevelopment or subdivision language in the listing
- no clear verified frontage/access advantage
- no verified planning controls or overlay context in this pass

## What prevents a Proceed outcome
The following material checks remain unresolved:
1. current zone confirmation from council mapping/site report
2. local plan / precinct confirmation
3. minimum lot size / subdivision threshold confirmation
4. frontage/access suitability confirmation
5. flood overlay check
6. steep land / landslide check
7. biodiversity / waterways check
8. prior application history check

## Current assessment view
This site is useful as a calibration case because it shows the difference between:
- a lot that is simply larger than average, and
- a lot that has a clearer verified subdivision pathway.

At present it remains plausible, but not persuasive.

## Recommendation
- **Current recommendation:** Hold / verify
- **Why:** not enough verified planning support exists yet to justify a stronger recommendation
- **Commercial usefulness:** useful because it tests whether the MVP overstates generic larger-lot opportunities without hard planning evidence

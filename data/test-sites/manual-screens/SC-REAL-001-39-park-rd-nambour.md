# Manual Screen #1 — 39 Park Rd, Nambour

## Executive summary
Initial screen outcome: **Hold / verify**

Reason:
- The site is a credible candidate for the MVP because it is a real Sunshine Coast listing with a visible address, lot size, and explicit redevelopment language.
- However, the decisive planning inputs remain unverified in this pass.
- The listing's strongest claim — **medium density zoning** — is still a listing claim until matched to an authoritative council source.

## What is encouraging
- Sunshine Coast LGA location: yes
- Nambour suburb is relevant to the MVP search set
- Real listing with visible address and lot size
- **598sqm corner block** is a positive physical signal
- Listing explicitly claims medium density zoning and redevelopment potential

## What prevents a Proceed outcome
The following material checks remain unresolved:
1. current zone confirmation from council mapping/site report
2. local plan / precinct confirmation
3. minimum lot size / subdivision threshold confirmation
4. flood overlay check
5. steep land / landslide check
6. biodiversity / waterways check
7. prior application history check

## Current assessment view
This is a **good real-world test case** for the MVP because it shows exactly why listing-led optimism is not enough.

At screen level, the site is promising enough to warrant deeper verification, but not yet strong enough for a confident "Proceed" recommendation.

## Recommendation
- **Current recommendation:** Hold / verify
- **Why:** too many material unknowns remain
- **Commercial usefulness:** high as a test case because it pressures the MVP on zoning, overlays, and history rather than just lot size

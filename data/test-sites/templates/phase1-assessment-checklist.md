# Phase 1 Manual Assessment Checklist

## Before starting
- Confirm the site is in Sunshine Coast LGA.
- Confirm the use case is residential subdivision.
- Open the test-site register and assign the site ID.

## Minimum inputs to capture
1. Address / lot-plan
2. Gross site area
3. Current zone
4. Local plan / precinct
5. Minimum lot size reference
6. Frontage / access note
7. Flood note
8. Steep land / landslide note
9. Biodiversity / waterways note
10. Prior application history note

## Placeholder discipline
- If a planning threshold is not yet verified, set placeholder flags to true.
- Do not convert indicative thresholds into factual statements.
- Unknowns are acceptable; hidden unknowns are not.

## Recommendation discipline
- More than 3 material unknowns => cap at hold_verify.
- Verified fatal blocker => do_not_proceed.
- Keep recommendation consistent with evidence notes.
